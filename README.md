# UniCompass 🧭

> **ნუ დაგუგლავ შენს მომავალს. მოახდინე მისი სიმულაცია.**

**UniCompass** არის თანამედროვე, სრულფასოვანი (*Full-Stack*) საგანმანათლებლო პლატფორმა და ინტელექტუალური დამხმარე აბიტურიენტებისთვის. ნაცვლად სტანდარტული, მოსაწყენი უნივერსიტეტების სიებისა, აპლიკაცია მომხმარებელს სთავაზობს ინტერაქტიულ გლობალურ რუკას და პერსონალიზებულ თავსებადობის ალგორითმს.

სისტემა აანალიზებს სტუდენტის აკადემიურ მონაცემებს, ფინანსურ ბიუჯეტსა და კლასგარეშე აქტივობებს და უნივერსიტეტის მოთხოვნებთან შედარებით, რეალურ დროში ითვლის მიღების ალბათობას.

## ✨ მთავარი ფუნქციონალი

* **🗺️ გლობალური ინტერაქტიული რუკა:** Leaflet.js-ზე დაფუძნებული რუკა, რომელიც აჩვენებს მსოფლიოს წამყვან უნივერსიტეტებს.

* **🧠 ინტელექტუალური ალგორითმი (Matching Engine):** ბექენდზე მომუშავე სერვისი, რომელიც ითვლის მომხმარებლის პროფილისა და კონკრეტული საბაკალავრო/სამაგისტრო პროგრამის თავსებადობის პროცენტს. რუკაზე პინების ფერები (მწვანე, ყვითელი, წითელი) დინამიურად იცვლება ამ პროცენტის მიხედვით.

* **🔍 ნაკლოვანებების ანალიზი (Gap Analysis):** თუ სტუდენტის მონაცემები არ არის 100%-ით თავსებადი, სისტემა ავტომატურად აგენერირებს რჩევებს (მაგ: *"ამ პროგრამისთვის გჭირდებათ მინიმუმ 7.0 IELTS"*, ან *"თქვენს პროფილს აკლია 'ლიდერობის' და 'კვლევის' გამოცდილება"*).

* **👤 პროფილის მართვის Dashboard:** სრულყოფილი სამართავი პანელი, სადაც მომხმარებელს შეუძლია განაახლოს თავისი GPA, ენები, უნარები, დაამატოს უნივერსიტეტები ფავორიტებში და მართოს სამომავლო ამოცანების სია (To-Do Checklist).

* **🔒 უსაფრთხო ავტორიზაცია:** მრავალსაფეხურიანი სარეგისტრაციო ფორმა Email OTP (ერთჯერადი კოდის) ვერიფიკაციით და პაროლის აღდგენის უსაფრთხო ფუნქციონალით.

## 🛠️ გამოყენებული ტექნოლოგიები

### 💻 Frontend

* **HTML, CSS, JavaScript**

* **Leaflet.js & CartoDB** - რუკის რენდერინგისთვის.

* **Lucide Icons** - მინიმალისტური ვექტორული აიკონებისთვის.

### ⚙️ Backend & Security

* **Java & Spring Boot** - ძირითადი ბექენდ ფრეიმვორკი.

* **Spring Security & JWT** (JSON Web Tokens) - Stateless, დაცული API ენდპოინტებისთვის.

* **JavaMailSender** - OTP კოდების და პაროლის აღდგენის HTML მეილების გასაგზავნად.

### 🗄️ Database

* **PostgreSQL** - მთავარი მონაცემთა ბაზა.

* **Spring Data JPA (Hibernate)** - მონაცემთა მართვისთვის.

## 🚀 ლოკალურად გაშვების ინსტრუქცია (Setup Instructions)

**1. დააკლონირეთ რეპოზიტორი:**

git clone https://github.com/TazoB/UniCompass.git


**2. გახსენით პროექტი თქვენს IDE-ში.**

**3. `.env.example` ფაილის სახელი შეცვალეთ `.env`-ით და შეავსეთ თქვენი მონაცემებით:**

* მიუთითეთ თქვენი ლოკალური PostgreSQL-ის `url`, `username` და `password`.

**4. გაუშვით Spring Boot აპლიკაცია** (`UniCompassApplication.java`).

**5. ბაზის ინიციალიზაცია (Seeding):**
თუ მონაცემები ავტომატურად არ დაემატა, გაუშვით უკვე არსებული `.sql` ფაილები `resources` ფოლდერიდან.

**6. გახსენით ბრაუზერი:**
კოდი უნდა გაუშვათ ლოკალურ სერვერზე:

http://localhost:8080
